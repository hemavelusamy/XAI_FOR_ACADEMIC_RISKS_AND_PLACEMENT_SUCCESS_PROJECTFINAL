# Academic Placement Dashboard

## Project Description
An interpretable dashboard designed to predict student placement outcomes and identify academic risk factors early. It uses XGBoost for predictive modeling and SHAP for explainable AI (XAI) insights.

## Features
- Student Academic & Placement Tracking.
- XGBoost Model for Placement Prediction.
- XAI (SHAP) interpretation for transparent results.
- Proactive intervention for at-risk students.

## Technologies Used
- **Backend:** Django
- **Machine Learning:** Scikit-learn, XGBoost, SHAP
- **Database:** SQLite
- **Deployment:** GitHub

## How to Run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run the server: `python manage.py runserver`
